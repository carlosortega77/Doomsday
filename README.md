# DOOMSDAY · Dojo de Cálculo Mental

Aplicación web autocontenida (single-file HTML) para entrenar cálculo mental.

## Demo

https://carlosortega77.github.io/Doomsday/

## Uso local

Abre `index.html` en cualquier navegador moderno. No requiere instalación ni servidor.

## Stack

HTML + JS embebido. Cero dependencias externas.
